# Welcome to Pravaah

Pravaah is an AI-native workflow orchestration framework designed for building scalable backend systems through plugins, events, automation, and intelligent workflows.

## Core Features

- Plugin Architecture
- Dynamic CRUD Generation
- Event-Driven Automation
- AI Service Layer
- Workflow Orchestration
- Docker Deployment
- Async-First Design

## Philosophy

**Everything Flows.**

Pravaah treats applications as interconnected workflows rather than isolated API endpoints.
